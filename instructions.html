Dream Time Machine

Instructions

In order to use our program, you'll need atom, as well as a few extra packages. Check out the rest of this page for installation instructions for everything you need to get you started with your deep dream!

Installations

Run the following commands for each package to be able to run our program on your machine.

OpenCV: pip install opencv-python

Pillow: pip install Pillow

Librosa: pip install librosa

Pygame: pip install pygame

Mutagen: pip install mutagen

If you run into trouble with installing any of these, check out their documentation online.

Instructions for use

1. Download our repo.

2. Open up the file "Dream.py"

3. Navigate through the home screen to choose what settings you want to jam to.

4. Play around! You can move around and see what happens, check out what changes when you choose a different song, a different mode, etc.
